**LaetusBB**

Features:
* Purely SSR
* BBCode support


# Project Structure

```
laetusbb
│
└── laetusbb/
└── core/
└── apps/
│   └── thread/
│   └── user/
```


* core:
    * Forum models
    * entry point

* apps/thread:
    * Handle threads, posts, attachments, etc.

* apps/user:
    * [User profiles](https://docs.conan.io/en/latest/howtos/manage_gcc_abi.html)
